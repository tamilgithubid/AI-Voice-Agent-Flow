## рҹҺҜ Features

### Core
| Feature | Description |
|---------|-------------|
| рҹ—ЈпёҸ **Voice-First** | Full conversation via Web Speech API вҖ” speak to compose, confirm, and send |
| рҹӨ– **AI Intent Detection** | Groq LLM (llama-3.3-70b) understands "send email" vs "send WhatsApp" |
| рҹ“§ **Gmail SMTP** | Sends real emails via Nodemailer with Gmail App Password |
| рҹ’¬ **WhatsApp via Twilio** | Sends WhatsApp messages through Twilio sandbox/production |
| рҹ”Ҡ **Text-to-Speech** | Agent speaks back every response with natural voice selection |
| рҹҺЁ **Animated Pipeline** | Live SVG visualization of the AI agent pipeline (purple=email, green=WhatsApp) |

### v2.0 вҖ” Smart Features
| Feature | Description |
|---------|-------------|
| вңЁ **AI Smart Compose** | One-click AI enhancement вҖ” makes emails professional, WhatsApp casual |
| рҹҢҠ **Animated Sound Waves** | SVG equalizer bars + expanding ring animation when listening |
| рҹ“Ү **Contact Book** | Save contacts by voice вҖ” "save contact John", "send to John" |
| рҹ“Ӣ **Message Templates** | 6 pre-built templates вҖ” "use running late template" |
| рҹҺӨ **"Hey Agent" Wake Word** | Always-on passive listening вҖ” say "Hey Agent" to activate |
| рҹ”Ғ **Read Back** | "Read back my message" вҖ” agent reads your draft aloud before sending |
| вҢЁпёҸ **Text Fallback** | Type responses when voice isn't convenient |
| рҹ’Ө **Idle Nudge** | Agent prompts you if silent too long вҖ” with personality & jokes |

## рҹҡҖ Quick Start

### Prerequisites

- **Node.js 18+**
- **Groq API Key** вҶ’ [console.groq.com](https://console.groq.com)
- **Gmail App Password** вҶ’ [Google Account вҶ’ Security вҶ’ App Passwords](https://myaccount.google.com/apppasswords)
- **Twilio Account** (for WhatsApp) вҶ’ [twilio.com](https://www.twilio.com)

### 1. Clone & Install

```bash
git clone <your-repo-url>
cd Email-Agent

# Install backend
cd backend && npm install && cd ..

# Install frontend
cd frontend && npm install && cd ..
```

### 2. Configure Environment

```bash
cp .env.example .env
```

Edit `.env` with your credentials:

```env
# AI
GROQ_API_KEY=gsk_your_groq_api_key

# Gmail
GMAIL_USER=your.email@gmail.com
GMAIL_APP_PASSWORD=your_16_char_app_password

# Twilio WhatsApp
TWILIO_ACCOUNT_SID=ACxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
TWILIO_AUTH_TOKEN=your_auth_token
TWILIO_WHATSAPP_FROM=whatsapp:+14155238886
```

### 3. Start the App

```bash
# Terminal 1 вҖ” Backend
cd backend && node src/server.js

# Terminal 2 вҖ” Frontend
cd frontend && npm start
```

Open **http://localhost:3000** in Chrome (best speech recognition support).

## рҹҺӨ Voice Commands

### Conversation Flow
| Say This | What Happens |
|----------|--------------|
| `"WhatsApp"` or `"Email"` | Selects channel |
| `"john at gmail dot com"` | Sets email recipient (speech-cleaned) |
| `"plus 91 9876543210"` | Sets WhatsApp recipient |
| `"Yes"` / `"Send it"` | Confirms and sends |
| `"No"` / `"Edit"` | Opens edit options |
| `"Start over"` / `"Reset"` | Restarts the flow |

### Smart Commands
| Say This | What Happens |
|----------|--------------|
| `"Hey Agent"` | Wakes the agent from sleep |
| `"Read back"` | Agent reads your composed message aloud |
| `"Use running late template"` | Applies a pre-built template |
| `"List templates"` | Shows available templates |
| `"Save contact John"` | Saves current recipient as "John" |
| `"Send to John"` | Looks up John in contacts |

### Available Templates
| Template | Email Subject | Use Case |
|----------|--------------|----------|
| `running late` | Running Late | Quick delay notification |
| `leave request` | Leave Request | Formal leave application |
| `meeting reminder` | Meeting Reminder | Team meeting heads-up |
| `thank you` | Thank You | Gratitude message |
| `follow up` | Follow Up | Check-in on pending items |
| `sick leave` | Sick Leave Notification | Sick day notification |

## рҹ”’ Security

| Layer | Protection |
|-------|-----------|
| **Rate Limiting** | 30 requests/minute per IP |
| **Input Validation** | Joi schemas on all endpoints |
| **Security Headers** | Helmet middleware |
| **CORS** | Restricted to frontend origin |
| **Credentials** | All secrets in `.env` (never in code) |
| **Confirmation** | User must confirm before any message is sent |

## рҹҗӣ Troubleshooting

| Issue | Solution |
|-------|----------|
| Microphone not working | Use **Chrome**, allow mic permission, check `chrome://settings/content/microphone` |
| Voice not speaking | Click anywhere on the page first (browser requires user gesture for audio) |
| "Hey Agent" not working | Complete a conversation first вҖ” wake word activates after flow ends |
| WhatsApp not delivered | Join Twilio sandbox: send `join <your-code>` to `+14155238886` on WhatsApp |
| Gmail auth error | Use an **App Password**, not your regular password. Enable 2FA first. |
| CORS error | Verify `FRONTEND_URL=http://localhost:3000` in `.env` |
| Groq timeout | Check API key at [console.groq.com](https://console.groq.com) |
